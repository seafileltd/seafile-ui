
## About

This is a package for developing Seafile UI components.

## Development

Initialization

```
npm install
sudo pip install jinja2
```

Build css/html

```
npm build
```

Run dev server

```
npm start
```

Visit `127.0.0.1:3000`.


## Directory structure

`src/seafile-ui`: seafile-ui Python module. It is needed for using `jinja2` to find HTML templates.
